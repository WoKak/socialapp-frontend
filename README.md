# socialapp-frontend

Frontend of socialapp - application with filtering offensive post feature realized by
separate application with neural network.

#### Technology stack:

* TypeScript 2.1.6
* Angular 4
* HTML
* CSS
* npm

#### To run this app:
1) Clone code
2) Install npm
3) ```npm install```

then you can start application with:

```
npm start
```
